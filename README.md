# WEB103 Prework - *Creatorverse*

Submitted by: **Aiswaryaa Velumani**

About this web app: **Creatorverse is a full-stack web application designed to manage and showcase content creators through a clean, intuitive interface. The application allows users to view, add, edit, and delete creator profiles while enforcing required validation to prevent incomplete or empty entries. It demonstrates full CRUD functionality integrated with a backend database and emphasizes user experience through centered layouts, consistent typography, and clear navigation. The project highlights practical use of modern frontend development with a real database connection and version control using GitHub.**

Time spent: **72** hours
